Media Queries

consists of a media type and zero or more expressions that check for the conditions of particular media features. http://dev.w3.org/csswg/css3-mediaqueries/#abstract

For Example

/* Default styles for small screens */

body {
  font-size : 76%;
}

#body-container {
  width : 95%;
}

@media only screen and (min-width: 40em) {...}

@media only screen and (min-width: 50em) {...}

@media only screen and (min-width: 64em) {...}

@media only screen and (min-width: 82.6em) {...}

@media only screen and (min-width: 115em) {...}
